# SaveMe

A tkinter GUI based adventure game on Python. 
Take the role of Aaron and save yourself by solving Quests and see where this leads you to!

## REQUIREMENTS
- **Python 3.x (preferably 3.6 or higher)**
- Install **Pillow(PIL)** via pip:
    ```bash
    pip install pillow
    ```
